# IstinaEndfieldAssistant

欢迎使用IstinaEndfieldAssistant！她能帮你：自动化据点物资券交易，刷理智关，完成每日，帝江号收菜，大地图稀有资源收菜......更多可用功能持续开发中，敬请期待。

## 运行方式

我们支持云运算与本地运算两种运行方式：

### 本地运算模式

**硬件要求：**
- 至少拥有一张至少16GB显存的英伟达显卡
- 处理器支持AVX2指令集（买得起16G显存卡的人CPU一般也支持了把）

**使用步骤：**
1. 下载最新releases
2. 解压后双击点我启动软件，并在提示弹窗中选择下载
3. 完成下载后双击文件夹内start_server.bat

### 云运算模式

**网络要求：**
- 运行时网络环境较优

**使用步骤：**
1. 下载最新releases
2. 解压后双击点我启动软件，并在提示弹窗中选择跳过

**数据使用声明：**
使用云数据代表您已允许我们处理并在保证您隐私的情况下存储您的各类数据。

## 开始使用

软件使用方式与MaaAssistant相似

1、（可选）选择 云服务 标签页，点击注册，输入用户名后即可完成注册，点击页面右上角的选项启用云服务
2、选择 开始代理 标签页
3、点击 添加任务 ，添加你希望实现的功能
